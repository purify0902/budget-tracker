# 1인사업자 가계예산 월별 추이

사업수입, 세금보관, 경비, 월급이체와 가계예산을 월별로 입력하고 추이를 보는 정적 웹페이지입니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 매월 `월별 입력`에 실제 금액을 입력합니다.
3. `월 데이터 저장`을 누릅니다.
4. `추이 보기`에서 보고 싶은 항목을 선택합니다.

데이터는 브라우저 안에 저장됩니다. 컴퓨터를 바꾸거나 브라우저 데이터를 삭제할 수 있다면, 상단의 `내보내기`로 백업 파일을 저장해두세요.

## GitHub Pages 배포

1. 이 폴더의 파일을 GitHub 저장소에 올립니다.
2. 저장소의 `Settings > Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. `main` 브랜치와 `/root` 폴더를 선택합니다.
5. 생성된 Pages 주소로 접속합니다.

## 파일 구성

- `index.html`: 화면 구조
- `styles.css`: 화면 디자인
- `app.js`: 입력 저장, 표, 차트, 내보내기 기능
