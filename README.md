# 1인사업자 가계예산 월별 추이

사업수입, 세금보관, 경비, 월급이체와 가계예산을 월별로 입력하고, 실제 지출 내역을 누적해서 월별 추이를 보는 정적 웹페이지입니다.

## 주요 기능

- 탭 메뉴로 화면 구분: 월별 예산, 가계부, 분석, 백업
- 월별 예산 입력
- 입력 중 자동 임시저장
- 가계부 지출 내역 입력
- 지출 분류 추가/삭제
- 지출 분류별 월간 요약
- 예산 대비 실제 지출 비교
- 월별 추이 차트
- JSON 전체 백업/복원
- 예산 CSV, 지출 CSV 다운로드

## 저장 방식

입력한 데이터는 현재 브라우저의 localStorage에 저장됩니다.

- 같은 PC와 같은 브라우저에서는 다시 열어도 유지됩니다.
- 다른 PC나 다른 브라우저에서는 자동으로 보이지 않습니다.
- 브라우저 사이트 데이터를 삭제하면 사라질 수 있습니다.

중요한 데이터는 상단의 `내보내기`로 JSON 백업을 저장해두세요.

## 사용 방법

1. `월별 예산` 탭에서 월별 예산을 입력합니다.
2. 입력 중인 내용은 자동 임시저장됩니다.
3. `월 데이터 저장`을 누르면 월별 예산 기록에 저장됩니다.
4. `가계부` 탭에서 날짜, 분류, 내용, 가격을 입력합니다.
5. 필요 없는 지출 분류는 `분류 관리`에서 삭제하고, 새 분류는 추가합니다.
6. `분석` 탭에서 선택한 월의 분류별 실제 지출을 확인합니다.
7. `백업` 탭에서 전체 데이터를 내보내거나 가져옵니다.

모든 금액은 예시값 없이 비워져 있으며, 사용자가 직접 입력합니다.

## GitHub Pages 배포

1. 이 폴더의 파일을 GitHub 저장소에 올립니다.
2. 저장소의 `Settings > Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. `main` 브랜치와 `/root` 폴더를 선택합니다.
5. 생성된 Pages 주소로 접속합니다.

## 수정 후 업로드

GitHub Desktop 기준:

1. 파일을 수정합니다.
2. GitHub Desktop에서 변경사항을 확인합니다.
3. Summary에 수정 내용을 적습니다.
4. `Commit to main`을 누릅니다.
5. `Push origin`을 누릅니다.
